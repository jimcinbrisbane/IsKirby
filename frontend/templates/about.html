About this project

Technology Used

Inspired by

Cloudflare's lava lamp wall

Issue

True randomness is hard to generate using computers alone. We are hoping to explore more ways to generate elements randomly to extend security.

Function

Our program explores a potential way to generate continuous sources for new cryptographic seed data on a large scale.

Locale

Our initial project aim was to set up a Raspberry Pi containing a [REDACTED] hosted server. The aforementioned Raspberry Pi would be connected to a livestream of Sydney Harbour and would take a photo of the sky from that livestream. Revision of our project goal now includes extended functionality to upload a photo which will generate a cryptographic key.
